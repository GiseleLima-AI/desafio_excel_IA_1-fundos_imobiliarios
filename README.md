# Desafio Excel + IA - Fundos Imobiliários
Aplicando meus conhecimentos em Excel para criar um Simulador de Investimentos de Fundos Imobiliários.


Com esse aplicativo, o usuário poderá inserir o valor de salário ou renda mensal para simular diversos cenários de investimento.



Caso não queira adotar o valor sugerido para aporte, tem a possibilidade de definir um outro valor.

Além de simular o período desejado ou obter o resultado de diversos períodos previamente sugeridos no aplicativo, o usuário também poderá identificar seu perfil de investidor (conservador, moderado ou agressivo) e ver a distribuição de seu investimento nos Fundos Imobiliários (FIIs).
